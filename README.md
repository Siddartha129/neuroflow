# NeuroFlow AI

Agentic document intelligence workspace built with MERN, JavaScript, local/free AI, and offline fallbacks.

## Structure

- `client/` - React 18 + Vite frontend
- `server/` - Express + MongoDB/Mongoose backend with in-memory fallback

## Development

```bash
npm run install:all
npm run dev
```

Frontend: `http://localhost:5173`

Backend: `http://localhost:3001`

Copy `.env.example` to `.env` or `server/.env` and adjust values as needed.
